## Suggested Test Files

For best testing results, include:
1. A short PDF (1-5 pages) - for quick tests
2. A medium PDF (10-20 pages) - for typical use
3. A technical document - to test domain knowledge
4. A multi-topic PDF - to test retrieval accuracy
